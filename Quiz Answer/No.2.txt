deep learning เป็น model ทางคณิตศาสตร์ที่ถูกคิดค้นและพัฒนามาอย่างยาวนาน จนในปัจจุบันนั้นมีการอธิบาย deep learning ด้วยทฤษฎีที่ชื่อว่า universal approximation theorem ว่าด้วยการนำ non linear function ที่ไม่ซับซ้อน มาประกอบกันเพื่อสร้าง function ที่ซับซ้อนจนสามารถใช้เป็นตัวแทนของ function ใด ๆ ได้ และด้วยความเร็วในการคำนวนของ computer ในปัจจุบัน จึงทำให้ deep learning สามารถจัดการกับปัญหา non-linear ที่ซับซ้อน และมีขนาดใหญ่ จนมีความโดดเด่นที่นำหน้า machine learning อื่นๆ
