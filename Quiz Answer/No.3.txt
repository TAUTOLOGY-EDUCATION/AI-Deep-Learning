supervised learning, unsupervised learning และ reinforcement learning มีความแตกต่างกันดังนี้
  >  supervised learning เป็น machine learning ที่อาศัยลักษณะของข้อมูลที่เป็น labeled data หรือข้อมูลที่ถูก label เอาไว้โดยอาศัยผู้เชียวชาญมาใช้ในการเรียนรู้ และนำ model มาใช้ในลักษณะของการพยากรณ์ label ของข้อมูลอื่น ๆ
  >  unsupervised learning จะเป็น machine learning ที่จะอาศัยลักษณะของข้อมูลที่เป็น unlabeled data หรือ ข้อมูลที่ไม่มีการทำ label เอาไว้ มาใช้ในการเรียนรู้ และนำ model มาใช้ในลักษณะของการจัดกลุ่ม, จัดการโครงสร้างข้อมูล หรือการหาความเชื่อมโยงกันของข้อมูล 
  >  reinforcement learning จะเป็น machine learning ที่ไม่อาศัยข้อมูล แต่จะอาศัยกฎและเงื่อนไข มาใช้ในการเรียนรู้ผ่านการลองผิดลองถูก เพื่อสร้างกฎเกณฑ์ในการตัดสินใจที่ดีที่สุด (policy) ที่สามารถนำไปสู่เป้าหมายได้
