การเพิ่ม hidden node และ hidden layer ส่งผลให้ model มีความซับซ้อนที่เพิ่มมากขึ้น แต่จะมีความแตกต่างกันตรงที่ การเพิ่ม hidden node คือการเพิ่มจำนวน function ย่อยๆ ในขณะที่การเพิ่ม hidden layer คือการเพิ่มจำนวนการผ่าน activation function ของ function ย่อยในชั้นก่อนหน้า ทำให้การเพิ่ม hidden layer สามารถเพิ่มความซับซ้อนของ model ได้มากกว่า แต่ในขณะเดียวกัน การเพิ่ม hidden node นั้นสามารถสร้าง function ที่หลากหลาย ซึ่งในการเพิ่ม hidden layer มีข้อจำกัดในส่วนนี้ 

ดังนั้นในการปรับโครงสร้างของ deep learning จะต้องให้ความสำคัญกับทั้ง จำนวน function ที่หลากหลาย และ จำนวนการผ่าน activation function ของ function เหล่านั้น
